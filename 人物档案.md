## 一、主角

### 林安

| 项目 | 内容 |
|---|---|
| **身份** | 高中生，家住康民街三巷4栋二单元402 |
| **家庭背景** | 父亲林大强收废品，家境贫寒，住城中村出租屋 |
| **出场状态** | 重感冒高烧39度多，喉咙嘶哑 |
| **伤势积累** | 肋骨（像是）断裂（第5章被父亲掀飞时）；右手大拇指脱臼（第20章主动脱铐）；右脚脚趾翻盖（第6章踢到402金属时）；手掌多处玻璃割伤（第25章制作武器）；以上伤势均靠抢来的医院药品自行处理（第29章） |
| **核心能力（金手指）** | 嗅觉/听觉增强——能在远距离闻出怪物身上特有的甜腥铁锈气味；能在黑暗中靠呼吸节奏判断怪物意图；第11章本人第一次困惑"说不清自己为什么能在隔着铁门的地方闻到这个"——能力来源未解，推测与接触402辐射有关 |
| **感染状态** | 全程低烧，对辐射变异有某种抵抗力，卷一未解释原因 |
| **性格弧线** | 初始：被动+自责+等待救援 → 中段：主动脱铐+第一次反杀 → 后期：冷静设局，不念旧情 → 卷末：有了明确方向（老船厂找宁医生） |
| **行动轨迹** | 家→警车→传染病医院B3→医院内逃亡→废土街道→五金店→居民楼天台→防暴车→断桥→商场地下→猛禽皮卡 |
| **结局** | 存活，驾黑色猛禽皮卡冲出商场，后排带着年轻守卫，目标：老船厂冷库找宁医生 |
| **装备（卷末）** | 消防斧（主武器）、折叠刀、黑哥的土枪（已夺取）、猛禽皮卡、车内药品/食物/备用油桶 |

## 九、核心道具

### 暗灰色金属（402放射源）

- 来源：林大强从北郊废弃化工厂铅盒里捡回
- 特征：拳头大，约30斤，表面发烫，有臭氧味，暗处会发青绿色光（作者注：是从内部发光，非反光）
- 当前位置：第6章被林安踢落，滚进402防盗门外防尘垫底下，截止卷末仍在原地
- **关键悬念**：梁主任死前写下的预警内容（「402门外防尘垫附近」）已永久失传；全城没有任何人知道放射源的确切位置，只有林安知道

## 十、重要背景设定备忘

| 设定 | 内容 |
|---|---|
| **感染机制** | 接触高能辐射（非病毒/非咬伤必须）——梁主任已证明；接触时间越长/越直接，发病越快；陈队被抓伤约十八小时失控，诊所男人仅同一候诊室二十分钟也感染 |
| **变异特征** | 高烧→怕光→极度口渴→盯住血液/嗅血→力量暴增→完全异化（不认人，只追猎物）→高阶变异体可长出骨刺（第34章） |
| **怪物行为规律** | 主要追声音和震动（第26-30章林安确认）；在黑暗中声音优先于视觉；静止无声的人可以近距离不被察觉；高阶骨刺型对声音反应减弱，有战术意识 |
| **封城状态** | 第38章桥梁被人为炸断，所有出城口封闭；第48章电台宣布「一级清洗令」——高热/嗜血/有攻击倾向者就地处置 |
| **林安的谎言** | 第7章警察问「家里有没有可疑物品」，林安说没有——书名的核心；放射源至今无人知道真实位置 |
