# 《末日危机》爆款重改作战手册（番茄向）

## 1) 定位与读者画像（positioning）

### 题材定位
- 赛道：末世生存 + 人性黑暗 + 轻异能（非系统）
- 核卖点：`唯一知情者谎言` + `零号辐射源近在咫尺` + `主角不完全变异成长`
- 差异化标签：不是“天降系统开挂”，是“信息差、环境战、心理战”赢命

### 核心读者画像
- 男频主力（18-35），偏爱“高压求生、杀伐果断、连续反杀”
- 对“节奏慢、纯氛围描写”耐受低；对“每章有推进”要求高
- 点击动机：标题刺激、章末悬念、主角阶段性变强

### 传播关键词（用于书名/简介/标题）
- 末日、全城沦陷、零号源、辐射、反杀、绝境、独狼、尸潮、爆燃、黑化

---

## 2) 问题分级（高/中/低）

### 高优先（直接影响留存）
1. 前10章爆点密度不足，铺垫偏多
2. 章尾钩子强度不稳定，部分章节“读完可停”
3. 标题偏文学化，点击指向不够强

### 中优先（影响追更）
1. 第11-20章压抑段过长，小胜利不足
2. 主角前期被动偏多，“主动布局”比例偏低
3. 长期对手线薄，阶段boss更替快

### 低优先（影响二次传播）
1. 可复用金句不足（缺“口号级”表达）
2. 战利品爽点展示不系统（获得-使用-收益闭环不完整）

---

## 3) 50章重构蓝图（structure-redesign）

## 总体节奏规则
- 每1章必须推进至少1个核心变量（信息、武器、关系、危险等级）
- 每3章必须有一次可感知高潮（爆发/反杀/大反转）
- 每5章必须出现一次“可量化成长”（装备升级/规则认知升级/感官进化）

### 第一弧（1-10）：隐性爆发 -> 城市点燃
- 目标：3章立世界观，5章打出首个不可逆灾变，10章完成“城市级威胁已启动”
- 爽点节奏：3/5/8/10章给强刺激
- 悬念链：金属异常 -> 父亲异变 -> 谎言错报 -> 现场污染扩散

### 第二弧（11-20）：官方失明 -> 主角自救决心
- 目标：把“被审讯”改成“被困中布局”
- 节奏修正：每2章加入1个小胜利（骗过审讯、提前感知、工具改造）
- 章末主线：从“等救援”切为“只能我自己动手”

### 第三弧（21-30）：真相擦肩 -> 医院越狱
- 目标：连续动作流，尽量减少静态解释段
- 爽点节奏：23断电、25首杀、27抢药、29法则觉醒、30踏入废土
- 悬念链：规则识别（听觉诱导）-> 资源争夺 -> 人性背刺 -> 主角黑化定型

### 第四弧（31-40）：街区猎杀 -> 出城绝望
- 目标：主角从“逃命者”升级为“猎手型求生者”
- 强化点：陷阱战、路线规划、装备迭代、怪物进化对抗
- 弧尾重锤：断桥封城，拉高世界观与绝望级别

### 第五弧（41-50）：地库围城 -> 卷末爆发
- 目标：人性势力战 + 尸潮利用 + 资源垄断反夺
- 强节奏：潜伏（43）-> 探雷（44）-> 反客为主（45）-> 引尸清场（46-47）-> 枭雄斩首（48）-> 战车破局（49-50）
- 卷终状态：角色身份从“幸存者”升级为“区域级掠食者”

## 4) 前20章重构模板（front20-template）

## 前10章（留存段）模板
- 第1章：异常必须可视化（温热、臭氧味、昆虫暴毙三选二）
- 第2章：父亲出现“非人眼神”并对血产生反应
- 第3章：误诊结束时埋“错误归因”
- 第4章：门内进食异声 + 人物拒绝报案（人性困局）
- 第5章：家庭惨剧 + 逃命 + 金属位置落点（门外垫下）
- 第6章：报警前后加入“自责谎言”内心锚
- 第7章：关键问答“可疑物品？” -> 主角说谎
- 第8章：警方制服惨烈，抓伤伏笔明确
- 第9章：封锁现场里“污染持续”
- 第10章：播下“全城级蔓延”明示

## 11-20章（过渡段）模板
- 11-12：审讯不是重复争辩，而是“主角观察系统漏洞”
- 13-14：扩散与调查错位并行，形成戏剧反差
- 15-16：医院防线失守，熟人变异冲击主角
- 17-18：给主角一次“错误复盘”，但同时埋下正确证据
- 19-20：从被动挨打转向主动脱困（工具、时机、路线）

## 章尾钩子模板（固定四型）
1. 动作钩：`门锁弹开了。`
2. 信息钩：`他终于知道，自己听到的不是幻觉。`
3. 反转钩：`他以为逃出来了，下一秒灯全灭。`
4. 倒计时钩：`留给他的时间，不到十秒。`

## 5) 标题系统（title-system）

## 标题公式
- 公式A：`事件 + 生死后果`
  - 例：我爸捡回一块铁，当晚全家变怪物
- 公式B：`身份 + 反常行为`
  - 例：刑警队长被咬后，也开始喝生水
- 公式C：`动作 + 结果`
  - 例：我踹碎药房玻璃，十秒抢完救命药
- 公式D：`规则 + 反杀`
  - 例：我发现它们只追声音，于是反杀整层怪物

## 标题规则
- 长度：12-20字优先（可读、可点、可传播）
- 禁忌：抽象词过多（如“盲区、暗流、复盘”单独做标题）
- 必含元素：至少一个“冲突词”（咬、杀、断、逃、抢、炸、反杀）

## 示例（原 -> 建议）
- 三十斤的铁疙瘩 -> 我爸捡回一块铁，当晚家里出事了
- 无效的真话 -> 我说了真话，他们却给我打镇定剂
- 冷血法则觉醒 -> 他推我喂尸，我反手把他踹下楼
- 绝望的断桥 -> 出城桥被炸断，这座城被放弃了

## 6) 角色线重构（character-arc）

### 主角林安（五段成长）
1. 受害者：内疚、恐惧、否认
2. 求生者：识别规则，开始自救
3. 执行者：能做艰难决策，愿承担代价
4. 布局者：利用人性与环境反杀
5. 掠食者：主动规划路线与资源版图

### 长期对手线（建议新增）
- 不改世界观前提下，补一条“制度型对手”：
  - 例如：某应急系统中的强硬清洗派（通过广播、通缉、封锁政策持续施压）
- 作用：把冲突从“打怪”扩展为“人与秩序博弈”

### 情感锚点（建议新增）
- 新增1个可持续互利角色（非恋爱也可）：
  - 医疗向：懂处理感染/药品交换
  - 技术向：车辆/电力/地图情报
- 作用：提升追更粘性，缓解长期单人线疲劳

### 主角金句池（可循环使用）
- 想活命，先学会闭嘴。
- 末日里，信息比子弹贵。
- 我不是最强的，但我一定是最难杀的。

## 7) 商业化爽点机制（固定植入）

### 每5章固定一轮“升级闭环”
1. 获得：药/武器/载具/情报
2. 适配：改装/训练/试错
3. 应用：实战验证
4. 收益：战果与地位变化

### 爽点三件套（交替出现）
- 反杀爽：以弱胜强、以少胜多
- 认知爽：先发现规则、先一步行动
- 战利品爽：拿到别人拿不到的关键资源

---

## 8) 分阶段执行路线与验证指标（execution-roadmap）

## 阶段A（1-3天）：策略定稿
- 产出：全书新标题清单、前20章重构提纲、章尾钩子库
- 验证：每章能否一句话说清“冲突+推进+钩子”

## 阶段B（4-10天）：前20章重写
- 目标：开篇留存最大化
- 验证指标：
  - 每章结尾可形成“强制翻页感”
  - 每3章至少1个高潮
  - 前10章累计至少4次强刺激点

## 阶段C（11-18天）：21-50章重写
- 目标：中后段追更稳定
- 验证指标：
  - 每5章1次明确成长闭环
  - 每10章1次世界观升级或大转折

## 阶段D（19-20天）：统一润色与商业校准
- 统一术语：辐射机制、变异等级、武器命名
- 统一人设：主角决策风格与语言习惯不漂移
- 校准目标：简介、书名、前3章风格高度一致

---

## 9) 取舍建议（避免重改翻车）
- 保留：你现在最强的“压迫感、血腥真实感、空间动作逻辑”
- 谨慎：不要为了爽点硬塞系统或超规格外挂
- 放弃：大段重复性内心复盘、无推进的氛围铺陈
- 原则：每章都回答一个问题——主角比上一章更接近“活下去”了吗？
